The style guide

Every token and component in the VibeLoom UI design system, shown in one place. Crimson, cool greys, light and dark. Flip the theme in the top-right corner to confirm every surface adapts — and that diagrams and code stay legible islands in both.

Color tokens

Every colour is a --vl-* custom property. The neutrals and brand values flip between light and dark; the diagram palette (c1–c6, cs) is intentionally theme-independent so SVG figures stay readable on a light canvas.

A standard paragraph of body copy. The quick brown fox jumps over the lazy dog, then files a well-scoped pull request. Inline elements include a text link, some bold emphasis, a little italic nuance, an inline code token, and a keyboard hint like Cmd K.

A .vl-lead paragraph carries the section's opening thought at a larger, muted size.

Constraints aren't the enemy of good design — they're the brief. A system that flips cleanly between light and dark was never an accident.

Ordered list

  1. Define the tokens first — they are the contract.
  2. Build components that consume only tokens.
  3. Let the theme switch do the rest.

Unordered list

  • Crimson accent, cool-grey neutrals.
  • Diagrams and code render as fixed light/dark islands.
  • Focus rings, selection colour, and markers all token-driven.

Code

A .vl-code block is a dark island in both themes. Syntax is hand-tagged with .vl-kw, .vl-st, .vl-fn, .vl-nm, .vl-nu, and .vl-cm.

# bound the agent before you trust it
from vibeloom import Agent, Sandbox

agent = Agent(
    model="claude-opus",
    max_steps=8,
    sandbox=Sandbox(network=False),
)

def run(task):
    return agent.solve(task, retries=2)

Figure + diagram

A .vl-figure is a light island in both themes. It carries a __head (title + __num), an inline SVG built from diagram primitives (.fill-*, .solid-*, .stroke-*, .edge, .edge-strong, .dg-t, .dg-ink, .dg-center), and a figcaption. Toggle the theme — the figure stays put.

Band

A numbered section banner: .vl-band with a __num chip and a __title. Always reads on the inverted surface.
